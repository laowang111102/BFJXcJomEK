# 企业自动化办公小脚本 python748

## 项目概述

本项目是一个基于 Python 的自动化办公脚本，主要用于提高企业人员的工作效率。脚本能够处理从 A 系统（如 ERP）导出的 Excel 文件，根据自定义的数据映射关系表和B系统的导入模板，自动化生成适用于B系统的导入数据。

## 技术栈

- Python
- Excel 文件处理库（例如：openpyxl、pandas）

## 功能模块

### 文件处理

- 支持自定义选择文件及工作表（sheet）

### 数据映射

- 支持自定义配置数据映射关系

### 结果导出

- 支持自定义导出 Excel 结果文件（用于导入B系统）

## 系统角色

- 用户：使用脚本进行文件转换操作
- 管理员：配置数据映射关系，维护脚本

## 运行环境

- Python 环境搭建（建议版本3.x）

## 部署步骤

1. 安装 Python 环境及依赖库
2. 配置数据映射关系表
3. 运行脚本，根据指引完成文件选择和数据转换

## 目录结构

```
- project_root/
  - src/
    - main.py
  - templates/
    - B_system_import_template.xlsx
  - mappings/
    - data_mapping.json
  - results/
```

## 核心亮点

- 简便的文件选择和配置过程
- 灵活的数据映射关系配置
- 自动生成适用于目标系统的导入文件，提升工作效率
- 脚本化和可定制的特性，适应不同的企业需求
